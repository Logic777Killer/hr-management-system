# АСКУ — Автоматизированная Система Кадрового Учета

> 🎓 Учебный проект по дисциплине «Системная и программная инженерия»  
> 🏢 РТУ МИРЭА, Институт ИТ, Кафедра МОСИТ

## 📋 О проекте
**АСКУ** — это веб-приложение для автоматизации процессов управления персоналом. Система позволяет вести учет сотрудников, управлять структурой компании, формировать отчеты и контролировать кадровые события в соответствии с требованиями 152-ФЗ о персональных данных.

## 🚀 Функциональные возможности
Система поддерживает следующие роли: **Администратор, HR-менеджер, Бухгалтер, Руководитель, Сотрудник**.

### 🔹 Для HR и Администратора:
- ✅ **Управление сотрудниками**: Добавление, редактирование профилей, история переводов.
- ✅ **Структура компании**: Создание компаний и отделов.
- ✅ **Кадровые процессы**: Оформление приема на работу, увольнений, назначений на должности.
- ✅ **Отчетность**: Генерация отчетов по различным параметрам (PDF/Excel).
- ✅ **Безопасность**: Настройка ролей и прав доступа.

### 🔹 Для Сотрудников:
- 📄 **Личный кабинет**: Просмотр и обновление контактных данных.
- 📅 **Расписание**: Просмотр графика работы.
- 📩 **Обратная связь**: Отправка запросов в техподдержку.
